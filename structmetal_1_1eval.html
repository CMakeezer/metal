metal::eval< expr, args > Struct Template Reference
Metal » Functional

#include <metal/functional/eval.hpp>

Description

template<template< typename...> class expr, typename... args>
struct metal::eval< expr, args >

Evaluates an arbitrary expression expr with arguments args.

Usage

For any expression expr and arguments args,

using result = eval<expr, args...>;
Semantics:
if expr is evaluable with arguments args, then equivalent to
struct result :
expr<args...>
{};
otherwise, equivalent to
struct result {};

Example

template<typename x>
struct except_void
{
struct type;
};
template<>
struct except_void<void>;
using r1 = metal::eval<except_void, int>;
static_assert(std::is_base_of<except_void<int>, r1>::value, "");
using r2 = metal::eval<except_void, void>; //identity<void> is incomplete
static_assert(!std::is_base_of<except_void<void>, r2>::value, "");
